# Project 2: Mango Project Management Dashboard

A project management dashboard helps project teams track projects by showing high-level metrics for project progress as well as project task status, budget, and anything else that offers real-time insight into a project and allows professionals and freelancers access to key data and deadlines for ongoing and upcoming projects.
## Description

This repository is where we build code for the Mango Project Management Dashboard.
